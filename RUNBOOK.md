# MHA TSH Preservation Runbook

This public runbook intentionally does not include private deployment
instructions or local environment details.

The repository keeps clean-room compatibility code, tests, compact protocol
metadata, and sanitized progress notes. Operational notes can expose private
environment details, so they should be kept off-repository by maintainers.

For public contribution work:

1. Use `PROGRESS.md` for the current compatibility status.
2. Use `preservation_server/README.md` for the server package overview.
3. Use `CONTRIBUTING.md` for repository rules and accepted contribution types.
4. Keep screenshots, logs, APKs, OBBs, extracted assets, signing material,
   credentials, and local deployment notes out of Git.

Maintainers with authorized private environments should store operational
procedures in a private note outside this repository.
