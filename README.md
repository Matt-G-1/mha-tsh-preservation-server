# MHA TSH Preservation Server

An unofficial, clean-room compatibility server for preserving the discontinued
Android client of *My Hero Academia: The Strongest Hero*.

The current prototype supports:

- Local bootstrap and guest login.
- Player provisioning and Honei Urban Area entry.
- A seven-character owned roster.
- A verified Death Arms demonstration spawn.
- Scene completion, guide, map, ping, and reconnect packet handling.

See the [project site](https://matt-g-1.github.io/mha-tsh-preservation-server/)
for status and contribution entry points. [PROGRESS.md](PROGRESS.md) tracks
verified behavior and remaining work. [RUNBOOK.md](RUNBOOK.md) explains why
private deployment and device setup notes are not published in this repository.

## Contributing

Protocol research, tests, documentation, and clean-room server implementation
are welcome. Read [CONTRIBUTING.md](CONTRIBUTING.md) before opening an issue or
pull request.

Do not upload original APKs, OBBs, extracted assets, signing keys, copyrighted
art, or credentials. This repository intentionally contains only
interoperability code, compact protocol metadata, tests, and documentation.
